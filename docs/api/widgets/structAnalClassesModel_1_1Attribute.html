AnalClassesModel::Attribute¶

struct Attribute¶

List entry below a class.

This roughly corresponds to attributes of r2 anal classes, which means it is not an attribute in the sense of a class member variable, but any kind of sub-info associated with the class. This struct in particular is used to provide a model for the list entries below a class.
